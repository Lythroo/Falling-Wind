# Falling Wind

**A simple mod that adds dynamic wind sounds while you're falling, using the elytra sound effect.**

---

## What it does

When you fall in Minecraft, you'll hear wind sounds that get louder the faster you fall. It uses the vanilla elytra sound, so it fits right in with the game.

- Volume changes based on how fast you're falling
- Client-side only - works on any server
- **Mod Menu support** for easy configuration

---

## Requirements

- Minecraft **1.21.11**
- [Fabric API](https://modrinth.com/mod/fabric-api)

---

## Great for

Parkour maps, skyblock, or just making falling more fun.
